How To Play

Zombie Classes

Catcher
Balanced class with strong combat ability.
Mimic
Can disguise as humans to trick survivors.
Phantom
Can turn invisible temporarily.
Bomber
Explodes to damage and infect humans.

Human Perks

Rebellious (Default)

Gain 1 armor point on spawn.

Doubtful

Carry 3 guns instead of 2. Third gun deals more damage.

Lover

Gain lifesteal when landing shots on zombies.

Safety

Spawn with 1 heart and 5 armor. Armor and hearts regenerate.

Vanity

Triple jump instead of double jump.

Leader

Faster movement speed and camouflage while crouching/sliding.

Hopeful

Regenerate health constantly and revive zombies with 2 vaccine shots per round.

Event Perks (Secret)

Original Doubt and Original Vanity are hidden event perks.

Host Your Own Server

Prerequisites and Information

  • NAT Loopback capable router
    • Non NAT Loopback routers do work but joining differs.
    • If you have CG NAT router you cannot host.
  • Router capable of port forwarding
    • Some routers require manual internal/external port setup.
  • Game Server Client
Server Client Download Example Port Forward GAME SERVER Internal Port: 7777 External Port: 7777 SERVER ADDR Internal Port: 7778 External Port: 7778

Ports You Must Forward

Game Connection: 2838 Server Arbitrator: 2840, 2841, 2842, 2843 Voice Chat: 2832 Sync Server: 2932 Gateway examples: 192.168.1.1 192.168.0.1 192.168.1.254

Server Configuration
